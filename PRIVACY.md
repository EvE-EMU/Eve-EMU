# Privacy policy

**Last updated:** May 11, 2026

This policy describes how the **eve-emu** project on GitHub relates to your personal data.

## We do not want, collect, or retain your data

The project maintainers **do not** operate a product or service that asks you to sign up, **do not** build user profiles from this repository, **do not** sell or share personal data for marketing, and **do not** maintain a central database of visitors or contributors tied to your identity for our own purposes.

We **do not** want your personal data beyond what is unavoidable when you use normal GitHub features (see below).

## GitHub

If you view this repository, file issues, open pull requests, or use other GitHub features, **GitHub** (Microsoft) processes information according to their policies, not ours. See: [GitHub Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement).

## Self‑hosted software (e.g. Discord bot)

Code in this repository may be **run by you or your organization** on your own systems. In that case:

- **We** (the open‑source project) still **do not** receive that runtime data.
- **You** (or whoever runs the bot) are responsible for what gets stored on that machine (for example local config or state files the software writes), and for complying with applicable law and with third‑party terms (such as Discord’s).

If you only clone or read the code and do not run it against real accounts, nothing described in the previous paragraph applies beyond ordinary GitHub activity.

## Changes

We may update this file occasionally to keep wording clear. Substantive changes will be reflected in the “Last updated” date at the top.

## Contact

For privacy questions about **this repository and project**, open a GitHub issue in this repo or use the contact methods your maintainers publish in the repository (for example `README` or org profile). We cannot answer questions about GitHub’s or Discord’s own services on their behalf.
